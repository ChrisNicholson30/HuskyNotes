# Welcome to Husky Notes 🐺

A native, **open-source** Markdown notebook for iPhone, iPad and Mac.
Every note is plain *CommonMark* — and it always belongs to you.

## Why you'll like it

- Plain Markdown files, never a locked-in binary format
- A beautiful, fully themeable editor
- Works offline first; your data is yours

## Get started in three steps

1. Tap the **+** button to create a note
2. Type some Markdown — the editor styles it live
3. Pick a theme in **Settings** to make it yours

## A quick checklist

- [x] Created my first note
- [x] Tried a heading and a list
- [ ] Explored the built-in themes
- [ ] Exported a note as `.md`

## Markdown you can use

Mix `inline code`, **bold**, *italic*, and ~~strikethrough~~ freely.

```swift
// Notes are styled with a TextKit 2 editor.
func greet(_ name: String) -> String {
    "Hello, \(name)! Welcome to Husky Notes."
}
```

> Reliability over cleverness — sync correctness beats every other feature.
> Your notes round-trip losslessly to plain Markdown, every time.

Tag notes inline like #welcome to find them fast later.

Happy writing! ✍️
